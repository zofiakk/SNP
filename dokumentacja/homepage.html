SNP

A Snakemake pipeline for detection of SNPs using tools from GATK

Zofia Kochańska, Małgorzata Kukiełka, Jakub Białecki, Julia Smolik

Functions and tools used

  • Sequence mapping
  • Marking of duplicates
  • Modification of readings
  • Searching for variants
  • Selection of variants for analysis
  • Filtering of variants
  • Annotation of variants

TUTORIAL

How to call the program

In order to run the program, you have to be in the directory where the script is located and enter the following command in the console:

snakemake --use conda
